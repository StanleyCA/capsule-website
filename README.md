# Capsule Landing Page

A polished static landing page for Capsule, a consumer app for saving outfits, discovering style inspiration, and getting personal recommendations.

## Overview

The page is designed around one primary action: downloading Capsule from the App Store. It uses focused product messaging, an app-style preview, concise benefit sections, and a final conversion panel.

## Local Development

Open `index.html` in a browser. No build step or package installation is required.

## Notes

- Replace the App Store URL in `index.html` if Capsule uses a different production listing.
- Replace the generated hero illustration or the CSS-based in-app preview with real app screenshots when available.
- Replace the launch privacy and terms pages with final policy copy before publishing.
- Keep social links hidden or removed until real destinations are ready.

## Browser Support

The page uses semantic HTML, modern CSS Grid/Flexbox, and small vanilla JavaScript enhancements for scroll state and reveal animations.
